Tool-Genesis Leaderboard

Benchmark results under two evaluation paradigms (Direct and Code-Agent). Metrics follow a four-level evaluation: L1 surface compliance, L2 semantic interface fidelity, L3 functional correctness, and L4 downstream task utility.

Best in Column
L1

Surface Compliance

Compliance checks format adherence; Exec. (Server Execution) verifies that tools can be loaded and run without errors.

L2

Semantic Interface Fidelity

Schema-F1 measures how well the generated tool interface matches the reference specification in terms of parameter names, types, and structure.

L3

Functional Correctness

UTsoft and UThard evaluate functional correctness via unit tests with soft (partial match) and hard (exact match) criteria.

L4

Downstream Task Utility

SR (Task Success Rate) measures whether the generated tool can actually be used by an agent to solve the original downstream task.
